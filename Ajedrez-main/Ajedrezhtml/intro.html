Proyecto de Ajedrez en Java

Bienvenido a la página introductoria de mi proyecto de ajedrez, desarrollado en Java durante los últimos 4 meses.

Descripción del Proyecto

Este proyecto tiene como objetivo crear un juego de ajedrez completamente funcional. Durante el desarrollo, se han abordado varias áreas clave, incluyendo la lógica del juego, la interfaz gráfica y la base de datos.

Asi es mi logica de todas las piezas en el ajedrez: if ((mF == 2 && mC == 1) || (mF == 1 && mC == 2))

He aprendido mucho sobre programación en Java, especialmente sobre IU y sobre todo de matrices.

Definiciones importantes del ajedrez:

Jaque
Una amenaza directa al rey del oponente.
Jaque mate
Una posición en la que el rey del oponente está en jaque y no puede escapar.
Ejemplo
El rey está en jaque y no tiene movimientos legales para escapar.
Consecuencia
El juego termina y el jugador que dio jaque mate gana (Emilio queria que en mi proyecto de java calculase todas las opciones posibles para que se acabe asi,lo que me hacia falta).
Tablero
El área de juego del ajedrez, compuesta por 64 casillas alternas de colores claros y oscuros.
Peón
La pieza de ajedrez más numerosa y de menor valor.
Movimientos
El peón se mueve hacia adelante pero captura en diagonal.
Promoción
Si un peón llega al otro extremo del tablero, puede ser promovido a otra pieza, usualmente una reina.
Enroque
Una jugada especial en la que el rey y una de las torres se mueven simultáneamente.
Enroque corto
El rey se mueve dos casillas hacia la torre y la torre se coloca al lado del rey.
Enroque largo
El rey se mueve dos casillas hacia la torre del otro lado y la torre se coloca al lado del rey.
